### 第10页：损失函数设计

**公式展示**：

**总损失**：
```
L_total = L_L1 + L_perceptual + λ × L_adversarial
```

**L1损失**：
```
L_L1 = mean(|SR - HR|)
作用：保证像素级相似度
```

**感知损失**：
```
L_perceptual = mean(|VGG(SR) - VGG(HR)|)
作用：保证语义和纹理相似
```

**对抗损失**：
```
L_adversarial = -log(D(SR))
作用：生成逼真细节
权重λ：0.005
```

### 第12页：评价指标

**三大指标**：

**PSNR（峰值信噪比）**
- 公式：10 × log₁₀(255² / MSE)
- 范围：20-40 dB，越大越好
- 特点：像素级相似度

**SSIM（结构相似度）**
- 公式：[亮度] × [对比度] × [结构]
- 范围：0-1，越大越好
- 特点：更符合人眼感受

**LPIPS（感知相似度）**
- 公式：||VGG(img1) - VGG(img2)||²
- 范围：0-1，越小越好
- 特点：最符合人眼感受
